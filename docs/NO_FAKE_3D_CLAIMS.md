# Règle de validation des générations 3D

Un workflow ne doit pas annoncer qu'un asset est « prêt pour la production » uniquement parce qu'un
GLB existe. Pour les robots, le rapport doit confirmer le rig, les clips d'animation, les matériaux,
les collisions et l'import Godot. Un contrôle visuel de l'aperçu reste obligatoire.

# Statut d'implémentation 3D

- TripoSR monoculaire : prototype historique, non destiné à la production.
- Générateur déterministe CRAWLER-7 : implémenté sur une branche, en attente de validation CI et
  inspection du rendu avant fusion.
- Hunyuan3D-2mv / TRELLIS : évalués, non activés faute de runner GPU compatible.
- UniRig : évalué, non activé faute de GPU et parce qu'il ne fournit pas les clips d'animation.

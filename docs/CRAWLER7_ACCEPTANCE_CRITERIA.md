# Critères d'acceptation — CRAWLER-7

Le modèle de production est accepté uniquement lorsque :

1. la silhouette générale reste basse, large et quadrupède ;
2. le corps, les articulations et les pieds sont des pièces séparées ;
3. les quatre capteurs rouges avant sont visibles ;
4. l'armature contient le corps et trois os par patte ;
5. les clips Idle, Walk, Run, Attack et Shutdown sont importables ;
6. les matériaux distinguent métal sombre, blindage brun, articulations et émission rouge ;
7. des collisions simples existent pour le corps et les pieds ;
8. le modèle est importé sans erreur dans Godot 4.7 ;
9. le triangle count et les dimensions sont enregistrés dans un rapport ;
10. un aperçu PNG est produit pour contrôle visuel avant intégration dans une scène.

Une réussite CI seule ne remplace pas le contrôle visuel du rendu. Le GLB n'est branché dans le
gameplay qu'après validation de l'aperçu et ouverture du fichier dans Blender ou Godot.

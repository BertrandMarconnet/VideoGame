# 3D production entry point

Pour CRAWLER-7, utiliser uniquement l'interface **Générer CRAWLER-7 production** ou l'action
**Generate production robot — rigged and animated**. Aucun upload d'image n'est nécessaire.

Le workflow historique TripoSR est conservé temporairement pour traçabilité, mais ses sorties ne sont
pas considérées comme des assets finaux. Il sera retiré après validation visuelle du premier GLB de
production.

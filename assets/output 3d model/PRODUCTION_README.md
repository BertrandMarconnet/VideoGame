# Production 3D models

Les fichiers suffixés `_production` sont générés par des scripts déterministes et doivent réussir les
contrôles de rig, animations, matériaux et import Godot avant commit.

Pour CRAWLER-7, utiliser `crawler_7_production.glb`. Le fichier historique `crawler_7.glb` provient
d'un essai TripoSR monoculaire et ne doit pas être intégré au jeu comme asset final.

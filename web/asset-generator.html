FONCTIONNEMENT

Choisissez les options, chargez vos références et cliquez sur le bouton final. Le formulaire GitHub structuré s'ouvre avec les champs préremplis. Après soumission, GitHub Actions génère le GLB, les textures, le rig, les animations, les zones de dégâts et le catalogue Godot.

IMAGES

Le navigateur ne peut pas transférer automatiquement tes fichiers locaux vers GitHub sans autorisation privée. Ils servent ici à vérifier le dossier. Dans GitHub, il faudra les sélectionner une seconde fois dans le champ « Images de référence ».

4 — CONTRÔLE

Le bouton ouvre le véritable formulaire GitHub à menus. Tu devras seulement recharger les mêmes images, cocher les confirmations puis soumettre.

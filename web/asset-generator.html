POURQUOI UNE VALIDATION GITHUB ?

Les fichiers locaux ne peuvent pas être transférés automatiquement vers GitHub par une autre page Web sans clé d'accès.

Le menu préremplit toutes les options. Dans le formulaire GitHub qui s'ouvre, il reste uniquement à sélectionner les mêmes images dans le bouton de téléversement, puis cliquer sur Submit new issue.
